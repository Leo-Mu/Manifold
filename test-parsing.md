## 常见问题

### 如何安装依赖？

运行以下命令安装项目依赖：
```bash
npm install
```

### 什么是TypeScript？

TypeScript是JavaScript的超集，添加了静态类型检查功能。它可以帮助开发者：
- 提前发现错误
- 提供更好的IDE支持
- 增强代码可维护性

### 为什么选择React？

React是一个流行的前端框架，具有以下优势：
1. 组件化开发
2. 虚拟DOM性能优化
3. 丰富的生态系统
4. 活跃的社区支持
